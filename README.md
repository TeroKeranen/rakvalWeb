# Rakval

## Description
Rakval is a comprehensive solution for efficient and effortless management and monitoring of construction sites. The application provides powerful tools for planning, executing, and overseeing construction projects, making it easier for construction professionals to manage their daily tasks. Rakval consists of three main components: a web application, a mobile application, and a backend service.

- **Web Application** is built with React, offering a dynamic and responsive user interface.
- **Mobile Application** is developed using React Native, enabling users to manage their construction projects anytime and anywhere.
- **Backend** is implemented using Node.js, ensuring efficient data processing and business logic execution.

## Technologies
- Frontend: React
- Mobile: React Native
- Backend: Node.js (Pre-configured and running)

## Getting Started

   ```bash
